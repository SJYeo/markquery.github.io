.nestedlink

link, hover

지정한 영역 전체에 링크 스타일을 만든다. 이미지 링크, 버튼 링크와 같은 HTML과는 겹치지 않도록 해야한다. 텍스트 요소에만 nested 되어야 한다. reference에서는 visited를 제외한다. 
개별 링크 스타일을 만들 때는 alink를 사용. 

// LessCSS 
.nestedlink(@LinkColor, @HoverColor,@BorderColor,@VisitedColor) {
          color: @LinkColor;
          text-decoration: none;
          border-bottom: 1px solid @LinkColor;
          padding-bottom: 2px;
  &:hover,
  &:focus {
          color: @HoverColor;
          text-decoration: none;
          border-bottom: 1px solid @BorderColor;
          padding-bottom: 2px;
  }
  &:visited {
          color: @VisitedColor;
          text-decoration: none;
          border-bottom: 1px solid @VisitedColor;
          padding-bottom: 2px; 
  }
}
a {
    .nestedlink-08c-005580 & {
    .nestedlink(#08c, #005580, #0d65a1,#b4a);
    }
}
