.text-size

px, em

텍스트에 사이즈가 적용된다. 
6 ~ 32 px까지, 0.375 ~ 2 em까지 출력되는 px와 em의 텍스트 사이즈는 동일하다. 
em의 소수점은 0.375일 경우 e375m, 1.125em일 경우 1e125m으로 표기한다. 권장 텍스트 사이즈는 16px와 1em이다.

.text-size-16 {
	font-size: 16px;
}

// LessCSS
@textsize-px: 6;
    .text-size-px-loop(@index) when (@index < 33) {
      .text-size-@{index} { font-size: unit(@index, px); }
      .text-size-px-loop(@index + 1);
    }
    // generate .text-size 6px ~ 32px by 1px
    .text-size-px-loop(@textsize-px);

.text-size

px, em

px 단위는 200px까지, em 단위는 40em까지 적용할 수 있다. 텍스트 사이즈를 크게 사용할 시 테블릿과 모바일 미디어쿼리에 사이즈의 적당한 값을 지정해주는 것이 좋다.

.text-size-40em {
	font-size: 40em;
}

// LessCSS
@textsize-em: 3;
    .text-size-em-loop(@index) when (@index < 41) {
      .text-size-@{index}em { font-size: unit(@index, em); }
      .text-size-em-loop(@index + 1 );
    }
    // generate .text-size 2em ~ 40em by 1em
    .text-size-em-loop(@textsize-em);
