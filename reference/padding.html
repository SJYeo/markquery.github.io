.padding

px

none

reference에서 제안하는 box-model의 사이즈는 10px 간격으로 10px부터 500px까지다. 
콘텐츠에 초점을 맞추면 디테일함은 요구되지 않는다. 특별한 콘텐츠 제작 시 디테일한 사이즈는 별도 표시한다. padding의 값은 10px씩 증가되어 500px까지다.

// lessCSS
@padding: 10;
    .padding-loop(@index) when (@index < 501) {
      .padding-@{index} { padding: unit(@index, px); }
      .padding-loop(@index + 10);
    }
    // generate .padding-10 ~ 500px by 10px
    .padding-loop(@padding);
