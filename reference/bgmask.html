.maskimg

url

bsmask는 background mask의 줄임말이다. 마스크 이미지는 배경색과 혼합하여 효과를 낼 수 있다. background-color 보다 먼저 css가 선언돼야 한다.  
네이밍은 subtlepatterns의 파일명을 따른다. 텍스트 마스크이미지 효과는 별도의 속성을 사용한다. 디폴트 이미지 경로는 /images/bgmask/ 이다. 

.maskimg-billie-holiday {
    background: url(/images/bgmask/billie-holiday.png);
  }
