Layout CSS는 bootstrap.css에 포함되어 있다. 유저의 커스텀 CSS 파일에 추가할 필요가 없다. bootstrap의 기본 Layout은 2개이며, markquery에서 추가된 Layout은 19개이다. 

Layout을 만들 때는 오직 <div> 태그만 사용된다. nomargin은 margin, padding 값을 없앤 레이아웃이고, pin은 container의 크기와 center, left, right 위치로 고정시키는 레이아웃을 뜻한다.

<div>

container

docs

// centered, fixed-width layouts
.container {  
  .container-fixed();
}

<div>

container-fluid

docs

// min- & max-width content
.container-fluid {
  padding-right: @gridGutterWidth;
  padding-left: @gridGutterWidth;
  .clearfix();
}

<div>

container-fluid-nomargin

docs

// min- & max-width content, nomargin
.container-fluid {
  padding: 0;
  .clearfix();
}

# ROW
HTML
class
example
note

Usage

<div>

.row-

docs page

layout의 세로 열를 markup하는 셀렉터다. bootstrap의 기본 열은 2개이며,  markquery에서는 row에 여백을 없애는 2개의 nomargin 열이 추가된다. 고정된 레이아웃을 만드는 경우가 아니라면 보통 .row-fluid 셀렉터가 많이 사용된다. row-fluid-nomargin은 하위 셀렉터인 span 행 셀렉터에 영향을 주어 span의 여백이 모두 nomargin으로 처리된다.

<div>

row

docs


.row {
  margin-left: @gridGutterWidth * -1;
  .clearfix();
}

<div>

row-nomargin

docs


.row-nomargin {
//  margin-left: @gridGutterWidth * -1;
  .clearfix();
}

  [class*="span"] {
  float: left;
  min-height: 1px; // prevent collapsing columns
  margin-left: 0;
//  margin-left: @gridGutterWidth;
}

<div>

row-fluid

docs


.row-fluid {
  width: 100%;
  .clearfix();
  [class*="span"] {
    .input-block-level();
    float: left;
    margin-left: @fluidGridGutterWidth;
    *margin-left: @fluidGridGutterWidth - (.5 / @gridRowWidth * 100 * 1%);
  }
  [class*="span"]:first-child {
  margin-left: 0;
}

<div>

row-fluid-nomargin

docs


.row-fluid {
  width: 100%;
  .clearfix();
  [class*="span"] {
    .input-block-level();
    float: left;
    margin-left: 0;
    *margin-left: 0;
//  margin-left: @fluidGridGutterWidth;
//  *margin-left: @fluidGridGutterWidth - (.5 / @gridRowWidth * 100 * 1%);
  }
  [class*="span"]:first-child {
  margin-left: 0;
}

레이아웃을 마크업할 때 row-fluid-nomargin 값을 사용하면 하위 셀렉터에 영향을 주어, span 행까지 여백이 사라진다. 여러가지 경우를 고려한 nomargin 셀렉터로 사용한다.

<div>

layout-nomargin

-

.layout-nomargin {
  margin-left: 0 !important;
  *margin-left: 0 !important;
}

<div>

span-nomargin

-

.span-nomargin [class*="span"] {
  margin-left: 0 !important;
  *margin-left: 0 !important;
}
