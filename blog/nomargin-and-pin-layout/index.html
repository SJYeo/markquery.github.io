Nomargin And Pin Layout

May 29 2013

Get A Peek Into The LessCSS Files

markquery에는 레이아웃에 필요한 nomargin과 pin이 추가되어 있습니다. bootstrap의 레이아웃은 경우에 따라서 제한적일 수 있습니다. 웹개발자라면 충분히 수정할 수 있지만, 콘텐츠 제작자까지 모두 활용할 수 없습니다. 레이아웃을 만들다보면 반드시 필요한 조건이 생깁니다. 이를테면 여백이 없는 gird가 필요한데, 그것을 nomargin이라고 이름 붙였습니다. 또한 콘텐츠의 어떤 한 면을 고정시켜야 하는 경우가 있다. 그것을 pin이라고 이름 붙였습니다.

Web Designers & developers

markquery source는 lessCSS입니다. less에 추가된 것은 nomargin과 pin 두 개의 selector입니다. 어떻게 활용되는지 docs에서 확인할 수 있습니다. nomargin은 어떻게 만들었을까요? bootstrap의 grid(span)에는 margin 값이 들어있습니다. 그 값을 없애면 layout의 크기가 달라집니다. layout의 사이즈를 유지한 채 grid(span)의 사이즈도 유지되어야 합니다. @media (min-width: 1200px) { }에서 .container와 .span12의 width 값은 1170px입니다. 여백값 없이 1170을 12로 나누면 97.5입니다. 이 값이 nomargin입니다. 이해하기 쉬울 것입니다. 오리지널에서는 @gridGutterWidth이 함께 계산됩니다. markquery의 nomargin에서는 다음과 같이 계산됩니다. 반올림 함수 ceil()는 없어도 됩니다. css valid는 소수점도 완벽하게 통과시킵니다.

// variables.less - nomargin 1200px min
@gridColumnWidth1200-nomargin: 97.5px;
@gridGutterWidth1200-nomargin: 0px;
@gridRowWidth1200-nomargin: (@gridColumns * @gridColumnWidth1200-nomargin) + (@gridGutterWidth1200-nomargin * (@gridColumns - 1));

// mixins.less file
.span (@columns) {
      width: ceil((@gridColumnWidth-nomargin * @columns) + (@gridGutterWidth-nomargin * (@columns - 1)));
}

nomargin과 pin 7개의 파일들에서 추적할 수 있습니다.

// container nomargin

.core (@gridColumnWidth-nomargin, @gridGutterWidth-nomargin) {

.spanX(@index - 1);

.offset@{index} { .offset(@index); }
.offsetX(@index - 1);
}
.offsetX (0) {}

.offset (@columns) {
margin-left: ceil((@gridColumnWidth-nomargin * @columns) + (@gridGutterWidth-nomargin * (@columns + 1)));
}

.span (@columns) {
width: ceil((@gridColumnWidth-nomargin * @columns) + (@gridGutterWidth-nomargin * (@columns - 1)));
}

margin-left: 0;
// margin-left: @gridGutterWidth * -1;
.clearfix();
}

.row-nomargin [class*="span"] {
float: left;
min-height: 1px; // prevent collapsing columns
margin-left: 0;
// margin-left: @gridGutterWidth;
}

// move media-query less. Set the container width, and override it for fixed navbars in media queries
.container-nomargin,
.navbar-static-top .container-nomargin,
.navbar-fixed-top .container-nomargin,
.navbar-fixed-bottom .container-nomargin { .span(@gridColumns); }

// generate .spanX and .offsetX
.spanX (@gridColumns);
.offsetX (@gridColumns);
}


// container-fluid nomargin

.fluid (@fluidGridColumnWidth-nomargin, @fluidGridGutterWidth-nomargin) {

.spanX (@index) when (@index > 0) {
.span@{index} { .span(@index); }
.spanX(@index - 1);
}
.spanX (0) {}

.offsetX (@index) when (@index > 0) {
.offset@{index} { .offset(@index); }
.offset@{index}:first-child { .offsetFirstChild(@index); }
.offsetX(@index - 1);
}
.offsetX (0) {}

.offset (@columns) {
margin-left: (@fluidGridColumnWidth-nomargin * @columns) + (@fluidGridGutterWidth-nomargin * (@columns - 1)) + (@fluidGridGutterWidth-nomargin*2);
*margin-left: (@fluidGridColumnWidth-nomargin * @columns) + (@fluidGridGutterWidth-nomargin * (@columns - 1)) - (.5 / @gridRowWidth-nomargin * 100 * 1%) + (@fluidGridGutterWidth-nomargin*2) - (.5 / @gridRowWidth-nomargin * 100 * 1%);
}

.offsetFirstChild (@columns) {
margin-left: (@fluidGridColumnWidth-nomargin * @columns) + (@fluidGridGutterWidth-nomargin * (@columns - 1)) + (@fluidGridGutterWidth-nomargin);
*margin-left: (@fluidGridColumnWidth-nomargin * @columns) + (@fluidGridGutterWidth-nomargin * (@columns - 1)) - (.5 / @gridRowWidth-nomargin * 100 * 1%) + @fluidGridGutterWidth-nomargin - (.5 / @gridRowWidth-nomargin * 100 * 1%);
}

.span (@columns) {
width: (@fluidGridColumnWidth-nomargin * @columns) + (@fluidGridGutterWidth-nomargin * (@columns - 1));
*width: (@fluidGridColumnWidth-nomargin * @columns) + (@fluidGridGutterWidth-nomargin * (@columns - 1)) - (.5 / @gridRowWidth-nomargin * 100 * 1%) + (@fluidGridGutterWidth-nomargin / 2 );
}

.row-fluid-nomargin {

.clearfix();
[class*="span"] {
.input-block-level();

*margin-left:0;

margin-left: 0;

.offsetX (@gridColumns);

.spanX(@index - 1);

.spanX (0) {}

.span(@columns) {

margin-left: @gridGutterWidth-nomargin;
}

// generate .spanX
.spanX (@gridColumns);

@gridRowWidth-nomargin: (@gridColumns * @gridColumnWidth-nomargin) + (@gridGutterWidth-nomargin * (@gridColumns - 1));

// nomargin 1200px min
@gridColumnWidth1200-nomargin: 97.5px;
@gridGutterWidth1200-nomargin: 0px;

// Fluid nomargin grid
