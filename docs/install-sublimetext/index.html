Sublime Text Editor

sublime text 에디터는 최고의 문서 편집기 중 하나이다. 마이크로소프트의 워드, Emacs, Vim, notepad++ 심지어 윈도우즈 기본 프로그램 메모장까지 널리 쓰이는 문서편집기들이 많다. 그 중에서 하나를 추천한다면, 오늘은 sublime text 2이다. 현재 3가 베타버전으로 개발되고 있다. 완벽한 무료 프로그램은 아니지만, 이 문서 편집기의 매력에 빠진다면 기꺼이 라이센스를 얻을 만하다.

Licenses are per-user, rather than per-machine

Installation

Sublime Text 2 Download : http://www.sublimetext.com/

OS X, 윈도우즈, 리눅스 모두 다운로드 가능하다. 다운로드 후 가장 먼저 할 일은 Sublime Package Control을 설치하는 것이다. sublime text는 유저가 원하는 거의 모든 패키지를 설치할 수 있는데, package control을 먼저 설치해두면 패키지 관리를 편리하게 해준다.

Sublime Package Control

sublime package control : http://wbond.net/sublime_packages/package_control

윈도우즈 기준으로 ctrol + ` 키를 입력하면 하단에 콘솔창이 나타난다. 명령어 라인에 다음의 코드를 입력한다. 콘솔창을 없애는 키는 Esc 버튼이다.

import urllib2,os; pf='Package Control.sublime-package'; ipp=sublime.installed_packages_path(); os.makedirs(ipp) if not os.path.exists(ipp) else None; urllib2.install_opener(urllib2.build_opener(urllib2.ProxyHandler())); open(os.path.join(ipp,pf),'wb').write(urllib2.urlopen('http://sublime.wbond.net/'+pf.replace(' ','%20')).read()); print('Please restart Sublime Text to finish installation')

입력 후에 설치가 완료되면, sublime text를 재시작 한다.

sublime text에서 패키지라는 용어는 플러그인과 같고, 패키지 설치 과정은 하단 상태바에 작게 표시된다.
