User Setting and Install IMESupport For 2 byte languages (Korean, Japanese, Chinese, etc.)

2byte 언어인 한국어, 일본어, 중국어 사용자는 설정을 추가해야 한다. 한국어의 경우로 설명한다. 먼저 네이버의 나눔폰트가 설치되어 있어야 한다.

Nanum Font

naver nanum font download : http://hangeul.naver.com/font

Preferences > Settings User 에 다음의 코드를 넣고 저장한다. 

{
	"color-scheme": "Packages/Color Scheme - Default/Monokai.tmTheme",
	"font-face": "Source Code pro",
	"font-size": 11,
	"ignored-packages":
	[
		"Vintage"
	]
}

IMESupport Package

Preferences > Package Control > install package에서 imesupport 검색 후 선택

단축키는 ctrl + shift + p이다. 사용자의 컴퓨터에 따라 윈도우즈 7 이상의 운영체계에서는 Microsoft Office IME 2010를 설치해줘야 하는 경우도 있을 것이다.

download Microsoft Office IME 2010 : http://www.microsoft.com/ko-kr/download/details.aspx?id=30390
