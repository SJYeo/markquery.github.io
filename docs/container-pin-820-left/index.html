Layout example : container-pin-820-left

container-pin-820-left는 최대 820px이고 왼쪽으로float: left 정렬된다. 유동 레이아웃이고, Media Queries 1200px를 기준으로 max-width: 820px로 설정되어 있다.

레이아웃 페이지는 데스크탑 디스플레이에서 확인하는 것이 좋다. 테블릿, 모바일에서는 각각의 레이아웃으로 보인다.

레이아웃 테스트 하는 방법

  • 브라우저를 마우스로 좌우로 줄이거나 늘이기
  • ctrl + - 브라우저 화면 축소
  • ctrl + + 브라우저 화면 확대
  • ctrl + 0 브라우저 화면 원래상태로 되돌리기

Warning!

@media (min-width: 1200px) { .container-pin-left { max-width: 820px; } }

<body>
  <div class="container-pin-820-left">
    ...
  </div>
</body>
