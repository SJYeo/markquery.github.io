Row example : row

row는 고정 열이다. 고정 레이아웃 container와 함께 사용한다. 이 문서와 같이 유동 레이아웃 속에 고정 레이아웃 container와 열 row 사용 할 때는 영역이 겹치는 것을 주의해야 한다.

row에는 bootstrap.css - margin-left: -20px 그리고 responsive.css - margin-left: -30px와 같은 속성 값이 있다.

이미지로 레이아웃과 열의 영역을 확인.

이 문서에서 container 레이아웃의 배경색은 옅은 회색, row 열의 배경색은 짙은 회색이다.

레이아웃 페이지는 데스크탑 디스플레이에서 확인하는 것이 좋다. 테블릿, 모바일에서는 각각의 레이아웃으로 보인다.

레이아웃 테스트 하는 방법

  • 브라우저를 마우스로 좌우로 줄이거나 늘이기
  • ctrl + - 브라우저 화면 축소
  • ctrl + + 브라우저 화면 확대
  • ctrl + 0 브라우저 화면 원래상태로 되돌리기

Warning!

고정 container와 row는 최상위 레이아웃으로만 사용한다. 유동형 레이아웃 속에 혼합하여 사용하지 않는다.

<body>
  <div class="container">
    <div class="row">
    ...
    </div>
  </div>
</body>
