What is Theme Package

sublime은 문서의 종류에 따라 테마를 선택할 수 있다. 문서의 종류에 맞춰 구문 강조가 예쁜 컬러로 표현된다. 구문강조 뿐만 아니라 문서의 배경화면 컬러도 변경할 수 있다. sublime text의 디폴트 테마는 Monokai인데, markdown 문서를 지원하지 않는다. Monokai 테마가 마음에 들고 계속 사용하길 원한다면 markdown 스타일을 추가해준다.

download : https://github.com/amelandri/Monokai-Customized

다운로드 받은 파일 중, Monokai Customized.tmTheme를 sublime text 디폴트 테마가 있는 폴더로 옮긴다.

Preferences > Browse Packages를 선택하면 패키지 폴더창이 뜬다. Color Scheme - Default 폴더에 넣으면 된다.

Select Color Scheme

Preferences > Color Scheme > 'Monokai'

Install Theme Packages

Tomorrow 테마를 추천한다. 많은 유저들의 사랑을 받는 테마다.

Preferences > Package Control > install package > tomorrow 검색 후 선택

단축키는 ctrl + shift + p이다.

또한 solarized 테마 패키지도 추천한다.

solarized theme : http://ethanschoonover.com/solarized
