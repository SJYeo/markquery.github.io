Install First Package, Markdown Preview

다시 sublime text를 실행한 후, 첫 번째 패키지를 설치해보자.

콘텐츠 마크업을 쉽게 해주는 markdown preview 패키지다. sublime text 메뉴에서 Preferences > Package Control을 선택하면 메뉴가 나타난다. 그 메뉴에서 install package를 선택하면 수많은 패키지 목록이 나온다. markdown을 검색해보자. markdown 패키지도 많이 있으니 하나씩 설치해보고 마음에 드는 패키지를 사용하면 된다. 이 문서에서는 markdown preview를 안내한다.

Installation

  • Preferences > Package Control > install package > markdown preview 검색 후 선택
  • 단축키는 ctrl + shift + p이다.

Markdown Preview Homepage : https://github.com/revolunet/sublimetext-markdown-preview

Usage

sublime text의 유용한 기능에는 Command Palette가 있다. 사용자 단축키다. markdown preview를 설치 후 이 패키지의 단축키를 지정해보자.

markdown preview는 4가지의 기능을 제공한다. preview라는 이름에 걸맞게 alt + m 단축키를 누르면 브라우저에서 HTML로 변환된 콘텐츠를 확인할 수 있으며, alt + h 단축키를 사용하면 HTML 코드를 얻을 수 있다.

사용자 단축키를 만드는 방법은 Preferences > Key Bindings - User에서 다음의 코드를 입력하고 저장한다(ctrl + s). 

[
{ "keys": ["alt+m"], "command": "markdown-preview", "args": {"target": "browser"} },
{ "keys": ["alt+h"], "command": "markdown-preview", "args": {"target": "sublime"} }
]

그리고 alt + m으로 미리보기를 할 때 브라우저 설정방법은 Preferences > Package settings > Markdown Previe > Setting Default 에서 "browser": "default",를 크롬, 파이어폭스 등의 설치 경로로 수정해주면 된다. 주의할 점은 반드시 \를 /로 수정해줘야 한다. 

example :    "browser": "C:/Users/HONG/AppData/Local/Google/Chrome/Application/chrome.exe",
