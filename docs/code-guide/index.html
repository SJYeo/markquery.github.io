code guide

코드 가이드에 따라 HTML과 CSS를 사용한다. HTML5를 기준으로 한다. 부득이하게 css의 네임이 변경될 수 있다. CSS의 속성은 유저의 사이트 CSS에 따라 적용되지 않을 수 있으며, 가능한 공통된 속성을 만든다. 코드의 수정은 사전동의 없이 진행하되, 릴리즈 노트에 명시한다.

html5 syntax

  • 언제나 HTML5 doctype 선언
  • nested elements는 TAB 키 한 번 입력
  • 속성값은 큰따옴표(double quotes " ") 사용
  • 속성 순서 role, class, id, data, href

css syntax

  • 대부분 속성명은 class로 사용
  • javascript 사용 시 id로 사용
  • 속성은 스페이스로 두 칸 입력
  • 속성의 끝에는 항상 쌍반점(semi-colon) ; 을 입력 - 압축 배포 시 삭제됨
  • 속성을 입력할 때는 :에 공백 한 칸 입력 - 압축 배포시 공백 없어짐
  • 속성은 한 줄 씩
  • 속성값 그룹을 지을 때는 한줄 씩 : , 콤마로 그룹핑
.selector,
.selector-secondary,
.selector[type="text"] {
padding: 15px;
margin: 0 0 15px;
background-color: rgba(0,0,0,.5);
box-shadow: 0 1px 2px #ccc, inset 0 1px 0 #fff;
}
  • 단, 하나의 속성이 사용되는 셀렉터는 한 줄로 처리
  • 약칭이 가능한 hex 컬러 코드는 #ffffff 대신에 #fff로
  • hex 컬러 코드는 소문자
  • 0px는 0으로
.text-color-fff { color: #fff; }
.text-size-16 { font-size: 16px }

css naming

CSS의 속성을 보지 않고 의미를 이해할 수 있도록 네이밍

.text-color-fff { color: #fff; }
.text-bgcolor-fff { background-color: #fff; }
.text-lineheight-10 { line-height: 10px; }

em의 소수점은 0.375일 경우 e375m, 1.125em일 경우 1e125m으로 표기한다.

.text-size-e375m { font-size: .375em; }
.text-size-1e125m { font-size: 1.125em; }
