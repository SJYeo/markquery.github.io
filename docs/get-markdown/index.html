Get Markdown

마크다운으로 콘텐츠를 작성하고 HTML을 얻을 수 있는 방법은 여러가지가 있다. 온라인 마크다운, 데스크탑 프로그램, sublime text와 같은 에디터에서 markdown 패키지 등 유저들에게 맞는 방법을 찾으면 된다.

1. Online Markdwon

온라인 마크다운 중에는 dillinger를 추천한다. 예쁜 컬러를 선택할 수 있고, 작성과 동시에 output 결과를 확인할 수 있다. 오른쪽 상단 utilities에서 show HTML를 클릭하면 HTML을 얻을 수 있다. 파일로 export할 수도 있다.

dillinger Homepage : http://dillinger.io/

2. Desktop Program

데스크탑 프로그램으로 설치할 수 있는 MarkdownPad다. 마크다운 파일 관리에 장점이 있다.

MarkdownPad Homepage : http://markdownpad.com/

3. Sublime Text

텍스트 에디터 중에 sublime text를 추천한다. 무수히 많은 특징을 가지고 있어서 누구에게나 유용하게 사용될 것이다. sublime text에 대해서는 다음 문서에서 자세히 다룬다.

Sublime Text Homepage : http://www.sublimetext.com/
